# Power BI MCQ Skills Assessment

A practical Power BI multiple-choice quiz webpage covering Basic (B1), Intermediate (B2), and Advanced (B3) levels.

Live site: https://shubhankarreddy.github.io/power-bi-mcq/

## Contents

- 25 questions per level
- Industry-relevant Power BI scenarios
- Immediate feedback with explanations
- Progress tracking and topic breakdown

## Levels

- Basic (B1): Power BI fundamentals, Desktop, data sources, imports, transformations, visuals, slicers, dashboards, and publishing
- Intermediate (B2): Power Query, modelling, relationships, DAX, time intelligence, advanced visuals, drill actions, bookmarks, RLS, and workspaces
- Advanced (B3): advanced DAX, ranking, variables, advanced modelling, composite models, Python/R, paginated reports, embedded analytics, dataflows, deployment pipelines, performance, governance, and Fabric
